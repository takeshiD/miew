# miew
miew is markdown previewer.

## Features

miew has several great features.
- selectable paser, renderer, draweer
- every module is avilability reused
- support mermaid-syntax
```mermaid
sequenceDiagram
	Alice -> Bob: Hello Bob!
	Bob -> Alice: Hello Alice
```
- support katex-syntax
$$
f(x) = 1 + x / ( 1 + x)
$$
